# second-rep
Программа ищет самую длинную строку в списке, а затем в этой строке находит самую длинную подстроку без повторяющихся символов.
Функция check_input() проверяет, является ли элемент строкой.
Функция get_substring() ищет самую длинную подстроку без повторений.
Функция main() обрабатывает список и выводит результат.
Пример использования:
main([[0, 4, 8], 'abca', 7, 'aaaaa'])
Вывод: 3, 'a'
